LES MÉTADONNÉES D'UNE IMAGE NUMÉRIQUE



Les métadonnées d'une image numérique sont des données "cachées" que décrivent les sources de l'image. Lorsqu'on les extrait, on es capable de connaître le type d'appareil utilisé pour produire l’image numérique, le lieu où elle a été prise, etc…




On peut les extraire grâce à des sites comme celui ci: https://exifmeta.com/
